# AI-Assisted SOC

This folder will contain experiments showing how AI can support SOC analyst workflows.

Planned projects:

* AI-assisted alert summarisation
* AI-assisted phishing email triage
* AI-generated incident report draft with human validation
* Prompt injection awareness for SOC workflows
* Limitations of AI in security operations

Important principle:

AI can support SOC work, but it should not replace human validation. Security decisions must be reviewed by a human analyst.
