# Cloud Security Labs

This folder will contain cloud security labs focused on AWS and cloud monitoring.

Planned labs:

* AWS IAM least privilege review
* S3 public access review
* CloudTrail monitoring
* GuardDuty alert review
* Cloud security incident response scenario

Each lab will include:

* Objective
* Tools used
* Steps performed
* Screenshots or command examples
* Security findings
* Remediation recommendations
* Lessons learned
